# 💧 jQuery Essentials - Full Course

This repository contains my step-by-step progress in learning **jQuery**, the most popular fast and concise JavaScript library. It covers everything from basic selectors to advanced AJAX implementation, focusing on the principle: *"Write Less, Do More."*

## 🚀 Learning Goals

In this repository, I am documenting the following skills:

* **Selectors:** Efficiently targeting elements using the `$` syntax.
* **Events:** Handling user interactions like `click`, `on`, `hover`, and `change`.
* **DOM Manipulation:** Easily adding, removing, and modifying HTML elements and attributes.
* **Effects & Animations:** Creating smooth transitions with `hide`, `show`, `slide`, and `animate`.
* **AJAX:** Performing asynchronous HTTP requests to fetch data from servers without refreshing the page.
* **jQuery UI:** Using pre-built widgets and interactions.

## 📂 Folder Structure
The modules are organized as follows:
1.  **jQuery Setup** - Installation and CDN integration.
2.  **Selectors & Filters** - Selecting elements with ease.
3.  **Event Handling** - Managing mouse and keyboard events.
4.  **Effects & Chaining** - Building interactive animations.
5.  **AJAX with jQuery** - Getting and Posting data.
